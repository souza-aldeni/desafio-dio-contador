# Desafio Dio Contador

No primeiro desafio de aprendizado JS, foi proposto aos alunos, usar a integração entre o JS e o HTML para um simples contador, com dois botões (um para somar e outro para diminuir) e uma areá de texto para apresentar o resultado.
Para este desafio, estou usando os seguintes recursos:
 - VS Code;
 - HTML/CSS3;
 - JavaScript;
 - Git/GitHub para apresentação dos resultados;
